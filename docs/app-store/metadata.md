## Short Description (80 chars)
AI가 도와주는 건강한 커플 대화. 갈등을 이해로, 오해를 소통으로 바꿔보세요.

## Full Description (Korean)
커플AI는 AI 기술을 활용하여 커플 간의 소통을 개선하는 관계 도우미 앱입니다.

### 주요 기능
- **AI 리프레이밍**: 갈등 상황을 양쪽 관점에서 분석하고 건설적인 대화 방법 제안
- **대화 녹음 & 분석**: 실제 대화를 녹음하고 AI가 패턴을 분석
- **쿨다운 타이머**: 감정이 격해졌을 때 호흡 가이드와 함께 진정 시간 제공
- **매일 대화 주제**: 매일 새로운 대화 주제로 소통 습관 만들기
- **주간 인사이트**: 대화 패턴 분석과 개선 방향 제안
- **파트너 공유**: 리프레이밍 결과를 파트너와 안전하게 공유

### 개인정보 보호
- 모든 대화 내용 암호화 (Fernet 암호화)
- 녹음 파일은 분석 후 자동 삭제
- PIPA(개인정보보호법) 준수
- 데이터 내보내기 및 계정 삭제 지원
