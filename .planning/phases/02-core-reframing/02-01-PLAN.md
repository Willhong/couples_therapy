Create new Django app `chat` in backend/apps/ with models per RESEARCH.md:

**Models (models.py):**

1. `Conversation` - A chat thread
   - id: UUIDField primary_key
   - user: ForeignKey to User (CASCADE)
   - couple: ForeignKey to Couple (CASCADE, null=True for solo users)
   - title: CharField(200) blank=True (auto-generated from first message)
   - is_active: BooleanField default=True
   - created_at, updated_at timestamps
   - Meta: ordering=['-updated_at'], db_table='conversations'

2. `Message` - Individual message
   - id: UUIDField primary_key
   - conversation: ForeignKey to Conversation (CASCADE, related_name='messages')
   - role: CharField choices=['user', 'assistant', 'system']
   - content: EncryptedTextField (use fernet_fields like consents app)
   - has_reframing: BooleanField default=False
   - reframing_data: JSONField null=True (structured reframing sections)
   - token_count: IntegerField null=True (for context management)
   - created_at timestamp
   - Meta: ordering=['created_at'], db_table='messages'
   - Property: is_user -> returns role == 'user'

3. `ConversationSummary` - Rolling summary for context management
   - id: UUIDField primary_key
   - conversation: ForeignKey to Conversation (CASCADE, related_name='summaries')
   - summary_text: EncryptedTextField
   - message_count: IntegerField (number of messages summarized)
   - created_at timestamp
   - Meta: ordering=['-created_at'], db_table='conversation_summaries'

4. `SharedReframing` - Reframing shared with partner
   - id: UUIDField primary_key
   - message: ForeignKey to Message (CASCADE, related_name='shares')
   - shared_by: ForeignKey to User (CASCADE, related_name='shared_reframings')
   - shared_with: ForeignKey to User (CASCADE, related_name='received_reframings')
   - privacy_level: CharField choices=['full', 'summary', 'none'] default='full'
   - partner_response: EncryptedTextField null=True
   - shared_at timestamp
   - Meta: db_table='shared_reframings'

Import UUID from uuid module. Import EncryptedTextField from fernet_fields.fields.

**Serializers (serializers.py):**
1. MessageSerializer
   - Fields: id, role, content, has_reframing, reframing_data, created_at
   - Read-only: id, created_at
   - content shown only if user owns conversation

2. ConversationSerializer
   - Fields: id, title, is_active, created_at, updated_at, message_count
   - message_count: SerializerMethodField (count of messages)

3. ConversationDetailSerializer
   - Extends ConversationSerializer
   - messages: MessageSerializer(many=True, read_only=True)
   - Include pagination for messages (latest 50)

4. SharedReframingSerializer
   - Fields: id, message_id, privacy_level, shared_at, partner_response
   - Nested: message (MessageSerializer read-only)

**Views (views.py):**
1. ConversationViewSet
   - list: Return user's conversations (filter by user=request.user)
   - retrieve: Return conversation with messages (verify ownership)
   - create: Create new conversation for user
   - Queryset filtered to user's conversations only

2. MessageViewSet (nested under conversation)
   - list: Return messages for conversation
   - create: Create user message (role='user' enforced)
   - No update/delete (per CONTEXT.md - messages permanent)

3. SharedReframingViewSet
   - list: Return received shares (filter shared_with=request.user)
   - create: Share a reframing with partner (validate couple connection)

**URLs (urls.py):**
- /conversations/ - ConversationViewSet
- /conversations/{id}/messages/ - MessageViewSet
- /shared/ - SharedReframingViewSet
