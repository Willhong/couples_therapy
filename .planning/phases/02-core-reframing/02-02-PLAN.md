<objective>
Replace the 5-call sequential LangGraph pipeline with a two-mode single-call architecture: the LLM decides whether to respond conversationally (chat mode) or produce a structured reframing analysis (reframing mode).

Purpose: Not every message should trigger a full reframing analysis. When a user is venting, providing incomplete info, or asking clarifying questions, the AI should respond conversationally -- empathize, organize input, ask follow-ups. When the conflict description is complete enough, the AI produces the full structured reframing. This makes the conversation feel natural while maintaining the analytical power when it matters. Performance is maintained at 1 LLM call per message (down from 5).

Output:
- Rewritten reframing_graph.py (two-mode single-call pipeline)
- New two-mode system prompt in system_prompts.py
- Updated views.py handling both response modes
- Minimal frontend changes in useChat.ts and useStreamingResponse.ts to handle the mode field
- Cleaned up urls.py (remove dead SSE route)
</objective>

   - No match: return None

2. **`async def run_reframing_pipeline(user_message: str, conversation_context: str = "") -> dict`**:
   - Step 1: `check_safety(user_message)`. If severe, return immediately:
     ```python
     {
         'mode': 'reframing',
         'final_response': safety_template['acknowledgment'],
         'analysis': safety_template['analysis'],
         'suggestions': safety_template['suggestions'],
         'is_abuse_detected': True,
         'safety_response': safety_template,
     }
     ```
   - Step 2: Build messages:
     - SystemMessage with TWO_MODE_SYSTEM_PROMPT
     - HumanMessage with: conversation context (if any) + `\n\n사용자 메시지: ` + user_message
   - Step 3: Call `get_chat_model(temperature=0.6, streaming=False)` and `await model.ainvoke(messages)`
   - Step 4: Parse JSON response using `_parse_json_response()`
   - Step 5: Check `parsed.get('mode')` to determine which path:

   **If mode == 'chat':**
   ```python
   {
       'mode': 'chat',
       'final_response': parsed.get('message', ''),
       'analysis': None,
       'suggestions': [],
       'is_abuse_detected': False,
       'safety_response': None,
   }
   ```

   **If mode == 'reframing':**
   ```python
   {
       'mode': 'reframing',
       'final_response': _build_readable_response(parsed),
       'analysis': parsed.get('analysis'),
       'suggestions': parsed.get('suggestions', []),
       'is_abuse_detected': False,
       'safety_response': None,
   }
   ```

   **If mode is missing or unrecognized** (fallback -- treat as chat):
   ```python
   {
       'mode': 'chat',
       'final_response': parsed.get('message', str(parsed)),
       'analysis': None,
       'suggestions': [],
       'is_abuse_detected': False,
       'safety_response': None,
   }
   ```
