Create `src/hooks/useRecordingConsent.ts`:

**State:**
- `consentRequest`: Current consent request object (or null)
- `myConsent`: boolean
- `partnerConsent`: boolean
- `status`: 'idle' | 'pending' | 'approved' | 'declined'

**Methods:**
- `initiateConsent()`:
  - Create recording_consents record with status 'pending'
  - Broadcast consent request to partner via Supabase realtime

- `giveConsent()`:
  - Update my consent in database
  - Broadcast consent status to partner
  - Check if both consented -> update status to 'approved'

- `denyConsent()`:
  - Update status to 'declined'
  - Broadcast denial to partner
  - Clean up consent request

**Realtime:**
- Subscribe to channel `consent:{coupleId}`
- Listen for 'consent_request', 'consent_given', 'consent_denied' events
- Update state based on partner's actions
